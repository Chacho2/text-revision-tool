# Text Revision Tool

A simple static website for revising your own writing for clarity, tone, and flow.

## Files
- index.html
- style.css
- script.js

## Publish on GitHub Pages
1. Create a new repository named: text-revision-tool
2. Upload these three files.
3. Go to Settings > Pages.
4. Under Branch, choose main and /root.
5. Save.
6. Your site should appear at:
   https://YOUR-USERNAME.github.io/text-revision-tool/
